# To update the deployed version
```
git add .
git commit -m "your message here"
npm run deploy
```

# Getting Started 
0. Make sure you have [Node installed](https://nodejs.org/en/download/). 
1. Clone the repository.
2. Go into the local repository.
3. Run `npm install`
4. Run `npm start`
